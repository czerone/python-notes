# Test and Debug Tutorial #

Python 測試寫法以及測試程式碼的方法

### Python Version ###

* python=3.6

### Packages ###

* pylint

### Content ###

* add_num.py : 簡易的加法程式
* test_add_num.py : 測試 add_num.py 的程式

### Usage ###

* 測試排版
```
$ pylint add_num.py
```
* 測試程式輸出
```
$ python test_add_num.py
```
* 測試程式的命名由 test_ 開頭
